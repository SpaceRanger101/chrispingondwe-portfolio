# Chrispin Gondwe Portfolio (Static Site)

This is a lightweight, mobile-first portfolio site with:
- Dark/Light mode toggle
- Hover animations on project cards
- Smooth-scrolling navigation
- Working contact form via EmailJS (client-side)

## Quick Start
Open `index.html` in your browser. For best results use a local server (e.g., IntelliJ built-in or VSCode Live Server).
